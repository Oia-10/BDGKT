# BDGKT
Source code for BDGKT

## Dataset
- [Physics](https://aic-fe.bnu.edu.cn/cgzs/kfsj/index.html)
- [Math](https://aic-fe.bnu.edu.cn/cgzs/kfsj/index.html)
- [ASSIST17](https://sites.google.com/view/assistmentsdatamining/dataset)
- [ASSIST09](https://sites.google.com/site/assistmentsdata/home/2009-2010-assistment-data)


## Requirements
- python >= 3.8
- pytorch >=2.1
- dgl >= 2.2
- numpy
- utils
- pandas
- sklearn


## Usage
python train.py --dataset {dataset} --user_num {user_num} --item_num {item_num} --skill_num {skill_num}
